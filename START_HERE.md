## 📁 檔案說明

### 📘 文件檔案
- **README.md** - 專案主文件,包含所有基本資訊
- **TEAM_COLLABORATION.md** - 團隊協作詳細指南

### 🚀 腳本檔案
- **start-windows.ps1** - Windows 自動啟動腳本
- **start-macos.sh** - macOS/Linux 自動啟動腳本
- **docker.sh** - Docker 快捷指令集合

### ⚙️ 配置檔案
- **.env.example** - 環境變數範本
- **docker-compose.yml** - Docker 服務配置
- **requirements.txt** - Python 依賴清單
